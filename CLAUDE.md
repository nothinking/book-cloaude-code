# 바이브코딩 클로드코드 완벽가이드 - 읽기 어시스턴트

## 프로젝트 목적
"바이브코딩 클로드코드 완벽가이드" 책을 읽으면서 내용을 체계적으로 정리하고 학습하기 위한 어시스턴트

## 어시스턴트 역할

### 이미지 처리
- 책 페이지 이미지에서 텍스트 추출 (OCR)
- 코드 블록, 표, 다이어그램 등 구조화된 콘텐츠 인식
- 한글/영문 혼합 텍스트 정확히 추출

### 텍스트 정리
- 추출된 텍스트를 마크다운 형식으로 구조화
- 핵심 개념, 명령어, 팁 등을 카테고리별로 분류
- 코드 예제는 적절한 언어 태그와 함께 코드 블록으로 포맷팅

## 정리 형식

### 챕터별 노트 구조
```markdown
# 챕터 제목

## 핵심 개념
- 주요 개념 1
- 주요 개념 2

## 명령어/단축키
| 명령어 | 설명 |
|--------|------|
| /help  | 도움말 표시 |

## 코드 예제
\`\`\`언어
코드 내용
\`\`\`

## 팁 & 노트
- 실용적인 팁
- 주의사항
```

## 파일 구조
```
gwangju/
├── CLAUDE.md          # 이 파일 (프로젝트 설정)
├── notes/             # 챕터별 노트
│   ├── chapter-01.md
│   ├── chapter-02.md
│   └── ...
└── summary.md         # 전체 요약
```

## 워크플로우
1. 사용자가 이미지 또는 텍스트 제공
2. 이미지인 경우: 텍스트 추출
3. 내용 분석 및 구조화
4. 적절한 노트 파일에 정리
5. 필요시 요약 업데이트
